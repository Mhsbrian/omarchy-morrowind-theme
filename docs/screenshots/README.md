# Screenshots

Add images here and reference them from the top-level `README.md`.

Suggested shots (keep them ~1600px wide, PNG or WebP):

| File | What to capture |
|------|-----------------|
| `desktop-dark.png` | Morrowind dark: bar + terminal + a wallpaper |
| `desktop-parchment.png` | Morrowind Parchment (light) |
| `terminal.png` | The 16-color palette in a terminal (`ls`, `btop`) |
| `lockscreen.png` | The optional themed lock screen |

Capture on Omarchy with:

```bash
omarchy capture screenshot          # region/window/screen picker
```

Then commit the files and update the image paths in `README.md`.
